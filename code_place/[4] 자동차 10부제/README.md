### 문제 설명
이번 달부터 양산시에서는 교통 혼잡을 막기 위해서 자동차 10부제를 시행한다. 자동차 10부제는 자동차 번호의 일의 자리 숫자와 날짜의 일의 자리 숫자가 일치하면 해당 자동차의 운행을 금지하는 것이다. 예를 들어, 자동차 번호의 일의 자리 숫자가 1이면 1일, 11일, 21일에 운행하지 못한다. 또한, 자동차 번호의 일의 자리 숫자가 0이면 10일, 20일, 30일에 운행하지 못한다.여러분들은 일일 경찰관이 되어 10부제를 위반하는 자동차의 대수를 세는 봉사활동을 하려고 한다. 날짜의 일의 자리 숫자가 int로 주어지고, 5대의 자동차 번호의 일의 자리 숫자가 한 칸씩 띄워진 하나의 String으로 주어졌을 때 10부제를 위반한 자동차의 대수를 출력하는 함수를 구현하시오.
### 입력
A에는 날짜의 일의 자리 숫자가, B에는 5대의 자동차 번호의 일의 자리 숫자가 주어진다. (A, B에서 주어진 일의 자리 숫자들은 모두 0에서 9까지의 정수 중 하나이다.)
### 출력
주어진 A, B 값을 이용해 10부제를 위반한 차량의 대수를 리턴한다.
### 예제 입력/출력
**예제 입력 1**
```
A = 1, B = '1 2 3 4 5'
```
**예제 출력 1**
```
1
```
**예제 입력 2**
```
A = 3, B ='1 2 3 5 3'
```
**예제 출력 2**
```
2
```
**예제 입력 3**
```
A = 5, B ='1 3 0 7 4'
```
**예제 출력 3**
```
0
```
