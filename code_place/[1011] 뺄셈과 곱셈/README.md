### 문제 설명
두 개의 정수를 입력받아서 뺄셈과 곱셈을 출력하는 프로그램을 작성하시오
### 입력
첫째 줄에 A와 B가 주어진다. (0 < A, B < 100)
### 출력
첫째 줄에 A-B의 결과를 출력한다.두번째 줄에 AXB의 결과를 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
5 3
```
**예제 출력 1**
```
2
15
```
