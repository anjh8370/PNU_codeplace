### 문제 설명
국어, 수학, 영어, 물리, 화학 의 5가지 시험이 있다. 당신의 다섯 과목의 성적을 입력하고 성적의 총합을 출력하는 프로그램을 작성하시오.
### 입력
성적은 100점 이하의 자연수이다.
### 출력
다섯 과목의 총합을 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
75
96
80 
97 
100
```
**예제 출력 1**
```
448
```
