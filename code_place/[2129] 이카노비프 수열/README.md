### 문제 설명
이카노비프 수열은 각 항이 항상 다음 두 항의 합과 같습니다.마지막 두 항은 항상 1입니다.예시)10개의 이카노비프 수열 :55, 34, 21, 13, 8, 5, 3, 2, 1, 1정수 값이 주어지면, 해당 크기만큼의 이카노비프 수열을 출력합니다.
### 입력
이카노비프 수열의 크기를 나타내는 정수 N을 입력합니다.(1 ≤ N ≤ 40)
### 출력
공백 하나로 구분된 이카노비프 수열을 입력한 크기만큼 출력합니다.
### 예제 입력/출력
**예제 입력 1**
```
3
```
**예제 출력 1**
```
2 1 1
```
**예제 입력 2**
```
5
```
**예제 출력 2**
```
5 3 2 1 1
```
**예제 입력 3**
```
10
```
**예제 출력 3**
```
55 34 21 13 8 5 3 2 1 1
```
