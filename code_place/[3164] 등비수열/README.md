### 문제 설명
어떤 규칙에 따라 수를 순서대로 나열한 것을 수열(sequences)이라고 한다.예를 들어2 8 32 128 512 2048 ... 은2부터 시작해 이전에 만든 수에 3을 곱해 다음 수를 만든 수열이다.이러한 것을 수학에서는 앞뒤 수들의 비율이 같다고 하여등비(비율이 같다의 한문 말) 수열이라고 한다. (등비수열 : geometric progression/sequence)시작 값(a), 등비(r), 몇 번째인지를 나타내는 정수(n)가 입력될 때n번째 수를 출력하는 프로그램을 만들어보자.
### 입력
시작 값(a), 등비의 값(r), 몇 번째 인지를 나타내는 정수(n)가공백을 두고 입력된다.(모두 0 ~ 10)
### 출력
n번째 수를 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
1 5 3
```
**예제 출력 1**
```
25
```
