### 문제 설명
리스트에서 원소를 지정하고지정한 원소를 맨 처음으로 삽입하는 함수를 작성해보자.
### 입력
첫 번째 줄에 리스트 원소를 입력한다.두 번째 줄에는 인덱스를 입력한다.리스트 원소는 모두 정수이며, 각 입력 숫자는 공백으로 구분된다.
### 출력
이동한 원소가 포함된 리스트를 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
1 2 3 4 5
4
```
**예제 출력 1**
```
[5, 1, 2, 3, 4]
```
### 제약 사항
- 시간 제한   1000ms
- 메모리 제한   256mb
