### 문제 설명
여는 괄호 ‘(’와 닫는 괄호 ‘)’ 만으로 구성된 문자열에서 아래의 조건을 만족하는 문자열을 완벽한 괄호 문자열이라 정의하겠다.() 는 완벽한 괄호 문자열이다어떤 문자열 x가 완벽한 괄호 문자열이라면, (x)도 완벽한 괄호 문자열이다.어떤 문자열 x와 y가 완벽한 괄호 문자열이라면, xy도 완벽한 괄호 문자열이다여닫이 괄호로 이루어진 문자열이 주어질 때 가장 긴 완벽한 괄호 문자열의 길이를 return 하시오.
### 입력
문자열의 길이 n (1 ≤ n ≤ 200,000)과 괄호로만 구성된 길이 n짜리 문자열 bstr이 주어진다.
### 출력
주어진 문자열에서 길이가 가장 길면서 완벽한 괄호 문자열의 길이를 출력한다. 완벽한 괄호 문자열을 찾을 수 없는 경우 0을 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
num : 5
bstr : (())(
```
**예제 출력 1**
```
4
```
**예제 입력 2**
```
num : 14
bstr : (()))()((()))(
```
**예제 출력 2**
```
8
```
### 힌트
첫 번째 샘플에서, 문자열 맨 처음 위치부터 4개를 잘라낸 (())가 가장 긴 올바른 괄호 문자열이다.두 번째 샘플에서, 문자열 6번째 위치부터 8개를 잘라낸 ()((()))가 가장 긴 올바른 괄호 문자열이다.
