### 문제 설명
첫째 줄에는 별 1개, 둘째 줄에는 별 2개, N번째 줄에는 별 N개를 찍는 프로그램을 작성하시오.단, 별은 오른쪽 정렬 시켜야 한다.
### 입력
첫째 줄에 자연수 N이 주어진다.
### 출력
첫째 줄부터 N번째 줄까지 차례대로 별을 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
5
```
**예제 출력 1**
```
*
   **
  ***
 ****
*****
```

- 메모리 제한   256mb
