### 문제 설명
여기에 원형 자물쇠가 있다. 이 원형 자물쇠는 1부터까지의 자연수가 순서대로 구성되어 있다. 아래 그림과 같이 1부터 8까지의 자연수로 구성된 원형 자물쇠가 있다고 하자.만약 비밀번호가 486이라고 가정했을 때, 현재 위치인 1에서 4로 이동하고 싶다면 시계방향으로 돌리거나 반시계 방향으로 돌리면 된다. 시계방향으로 돌리면 5번인 반면, 반시계 방향으로 돌리면 3번만에 맞출 수 있다. 그 다음, 4에서 8까지 시계방향으로 돌리면 4번, 반시계 방향으로 돌리면 4번이고, 8에서 6까지 돌리는데 시계방향으로 2번, 반시계 방향으로 6번이 움직여야 한다.즉, 비밀번호를 입력하는데 필요한 최소 횟수는 3 + 4 + 2 =  9번임을 알 수 있다.원형 자물쇠를 구성하는 숫자 개수인과 비밀번호가 주어졌을 때, 자물쇠를 푸는데 돌리는 최소 횟수를 구하여라. 단, 처음 원형 자물쇠가 주어졌을 때 시작하는 위치는 항상 1이다.
### 입력
첫 줄에 원형 자물쇠를 구성하는 숫자 중 최댓값을 의미하는 자연수과 비밀번호를 의미하는 문자열가 공백으로 구분되어 주어진다. 문자열의 길이는 1이상 10 이하이며, 문자열은 1부터까지의 자연수로만 구성된다.
### 출력
비밀번호를 맞추기 위해 돌려야 하는 최소 횟수를 출력한다.
### 예제 입력/출력
**예제 입력 1**
```
8 486
```
**예제 출력 1**
```
9
```
**예제 입력 2**
```
5 512143442
```
**예제 출력 2**
```
10
```
